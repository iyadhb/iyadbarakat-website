Privacy Policy

Last updated: July 2026

This site (iyadbarakat.com) is a personal site for Iyad Barakat's consulting work and his book, The Consulting Prompt Playbook. Here's what happens with your data when you visit.

What this site collects

Not much. This site uses Google Analytics (GA4) to understand traffic, and Kit to power the newsletter signup box. Here's the detail:

  • Google Analytics (GA4) collects standard web analytics data — pages viewed, approximate location, device and browser type — via cookies, so I can see how many people visit and what they're interested in. It doesn't identify you personally.
  • If you sign up for the newsletter, your email address is stored by Kit (the email provider I use) so I can send you updates. You can unsubscribe at any time via the link in any email.
  • The "Email" and "Contact" links open your own email client — nothing is transmitted to or stored by this site.
  • The Reader Bonus code page (/bonus/) checks your code entirely in your browser. No code, email, or other input you type there is sent anywhere or stored.
  • Links to Amazon and LinkedIn take you to those companies' own sites, which have their own privacy policies.

Hosting

This site is hosted on GitHub Pages. GitHub may collect standard server logs (such as IP address and browser type) as part of operating its hosting infrastructure. See GitHub's Privacy Statement for details.

Changes to this policy

If what this site collects ever changes, this page will be updated to reflect it, and the "Last updated" date above will change.
